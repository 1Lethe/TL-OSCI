# TL-OSC 项目 
使用OOP in C (OOC) 写法的示波器库，仿HAL库搭建

版本号     ：Beta V0.2
主要贡献者  :TongLewis 

## 设计特点：
1. 调用简单：使用仿HAL模式的方式设计，只需配置抽象寄存器即可调用，无需复杂配置。
2. 功能强大：可绘制波形与X轴，Y轴标尺以及标尺数字。
3. 代码规范：变量规范，宏丰富抽象层次高，错误处理方式规范，可读性强。
4. 移植方便：设计屏幕函数抽象层，使用不同屏幕只需修改一处代码，轻松移植。

## 项目结构

TL-OSC/
